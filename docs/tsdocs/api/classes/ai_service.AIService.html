    Class AIService

    AI Service for centralized model operations. Provides a unified interface for text generation across multiple providers. Handles model configuration, error handling, and response formatting.

    Example

    const result = await aiService.generateText({
      provider: "openai",
      modelId: "gpt-4",
      prompt: "Explain quantum computing",
      params: { temperature: 0.7, maxTokens: 500 }
    });
