    Non-streaming text generation action for simple chat completions. Creates or continues a conversation with message persistence. Used by the /api/chat-text HTTP endpoint.

    Param: args.prompt

    The user's input prompt

    Param: args.model

    Optional model override (provider-specific)

    Param: args.provider

    AI provider to use (default: "openrouter")

    Param: args.conversationId

    Optional existing conversation to continue

    Returns

    Generated text response with metadata

    Example

    const result = await ctx.runAction(api.chat.simpleChatAction, {
      prompt: "Explain TypeScript generics",
      provider: "openai",
      model: "gpt-4"
    });
    console.log(result.text);
    console.log(`Conversation ID: ${result.conversationId}`);
